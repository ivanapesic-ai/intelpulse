# Annex D: Maintenance & Support

| Category | Coverage |
|----------|----------|
| **Bug Fixes** | Resolution of defects in delivered functionality |
| **Security Patches** | Critical security updates and vulnerability fixes |
| **Platform Updates** | Compatibility updates for underlying infrastructure |
| **Minor Adjustments** | Small UI/UX tweaks (< 2 hours each, up to 10 total) |

## Data Refresh Cycle

### What's Included Per Cycle
- Pull latest data from all connected sources
- Process and normalize new records
- Recalculate maturity scores
- Update visualizations
- Quality assurance check
- Deployment to production

### Recommended Frequency
- **Quarterly** (every 3 months) for most use cases
- **Monthly** if tracking fast-moving sectors
- **On-demand** for specific client requests (additional cost)

### Additional Data Refresh Pricing
- €1,500 per cycle (includes all sources)
- €500 per single source refresh

## Service Level Agreement (SLA)

### Availability Target
- 99.5% uptime during business hours (Mon-Fri, 9:00-18:00 CET)
- Scheduled maintenance windows: Sundays 02:00-06:00 CET

### Exclusions
- Force majeure events
- Third-party service outages (Dealroom, PATSTAT)
- Issues caused by unauthorized modifications
- Browser compatibility issues for unsupported browsers

### Communication
- Incident notifications via email within 1 hour of detection
- Post-incident report within 5 business days for Critical issues
- Monthly status summary (Premium tier only)
